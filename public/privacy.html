📋 Εισαγωγή

Η παρούσα Πολιτική Απορρήτου περιγράφει τον τρόπο με τον οποίο η εφαρμογή Synodos Med (synodos-med.gr) χειρίζεται τα δεδομένα σας.

Σεβόμαστε απολύτως την ιδιωτικότητά σας. Η φιλοσοφία μας είναι Privacy-First: τα δεδομένα σας παραμένουν στη συσκευή σας.

📦 Δεδομένα που αποθηκεύει η εφαρμογή

Η εφαρμογή αποθηκεύει αποκλειστικά τοπικά στη συσκευή σας (μέσω localStorage) τα εξής:

  • Καταγραφές ζωτικών σημείων (πίεση, σφυγμοί, θερμοκρασία, SpO₂)
  • Υπενθυμίσεις φαρμάκων και ιστορικό λήψης
  • Προσωπικές σημειώσεις και σημειώσεις ημερολογίου
  • Καταστάσεις checklists (τσεκαρισμένα πεδία)
  • Προτιμήσεις εφαρμογής (π.χ. θέμα χρωμάτων)

✅ Κανένα από αυτά τα δεδομένα δεν αποστέλλεται σε εξωτερικό server. Δεν υπάρχει βάση δεδομένων μας. Αν διαγράψετε τα δεδομένα της εφαρμογής από τον browser σας, διαγράφονται οριστικά.

📊 Στατιστικά χρήσης (Analytics)

Η εφαρμογή δεν χρησιμοποιεί Google Analytics ή άλλο σύστημα παρακολούθησης χρηστών.

Ενδέχεται να συλλέγονται ανώνυμα συγκεντρωτικά στατιστικά μέσω της υποδομής Vercel (πλατφόρμα hosting), όπως:

  • Αριθμός αιτημάτων (requests) στον server
  • Γεωγραφική περιοχή πρόσβασης (σε επίπεδο χώρας)
  • Τύπος browser / λειτουργικό σύστημα

Αυτά τα δεδομένα είναι εντελώς ανώνυμα, δεν συνδέονται με καμία προσωπική ταυτότητα και χρησιμοποιούνται αποκλειστικά για τη βελτίωση της απόδοσης της εφαρμογής.

📲 Offline Ανταλλαγή Δεδομένων (P2P Handover)

Η εφαρμογή περιλαμβάνει λειτουργία συγχρονισμού δεδομένων κατά την αλλαγή βάρδιας (Handover) μεταξύ συνοδών. Η διαδικασία αυτή πραγματοποιείται αποκλειστικά offline μέσω σάρωσης κώδικα QR (QR Code scanning) απευθείας από τη μία συσκευή στην άλλη. Κατά τη διαδικασία αυτή, τα δεδομένα μεταφέρονται Peer-to-Peer χωρίς τη μεσολάβηση του διαδικτύου, κάποιου cloud server ή οποιασδήποτε τρίτης υπηρεσίας.

🍪 Cookies & Τεχνολογίες αποθήκευσης

Η εφαρμογή δεν χρησιμοποιεί cookies για tracking ή διαφημιστικούς σκοπούς.

Χρησιμοποιούμε τις εξής τεχνολογίες αποθήκευσης του browser σας:

  • localStorage — για τα δεδομένα σας (ζωτικά, σημειώσεις, υπενθυμίσεις)
  • Cache Storage — για την offline λειτουργία της εφαρμογής (PWA Service Worker)

Μπορείτε να διαγράψετε αυτά τα δεδομένα ανά πάσα στιγμή μέσω των ρυθμίσεων του browser σας ή μέσω της λειτουργίας «Εξαγωγή / Διαγραφή Δεδομένων» της εφαρμογής.

🔗 Τρίτες υπηρεσίες

Η εφαρμογή κάνει χρήση των παρακάτω εξωτερικών υπηρεσιών:

  • Google Fonts — φόρτωση γραμματοσειρών (Inter, Outfit). Η Google ενδέχεται να καταγράφει ανώνυμα αιτήματα. Πολιτική Google
  • Google Maps — σύνδεσμοι πλοήγησης (ανοίγουν εκτός εφαρμογής). Χρησιμοποιείται μόνο αν επιλέξετε να ανοίξετε χάρτη.
  • Vercel — πλατφόρμα hosting. Πολιτική Vercel

⚕️ Αποποίηση Ιατρικής Ευθύνης

⚠️ Η εφαρμογή Synodos Med δεν παρέχει ιατρική διάγνωση, θεραπεία ή ιατρικές συμβουλές.

Το περιεχόμενο της εφαρμογής έχει αποκλειστικά ενημερωτικό και οργανωτικό χαρακτήρα και δεν υποκαθιστά:

  • Τη γνώμη εξειδικευμένου ιατρού ή επαγγελματία υγείας
  • Οδηγίες νοσοκομείου ή κλινικής
  • Επίσημα ιατρικά πρωτόκολλα

Σε περίπτωση έκτακτης ανάγκης, καλέστε άμεσα ΕΚΑΒ 166 ή Ευρωπαϊκή γραμμή έκτακτης ανάγκης 112.

🔒 Τα δικαιώματά σας (GDPR)

Σύμφωνα με τον Γενικό Κανονισμό Προστασίας Δεδομένων (GDPR — Κανονισμός ΕΕ 2016/679), έχετε δικαίωμα:

  • Πρόσβασης — να γνωρίζετε τι δεδομένα αποθηκεύει η εφαρμογή
  • Διαγραφής — να διαγράψετε όλα τα δεδομένα σας (λειτουργία εξαγωγής/διαγραφής στην εφαρμογή)
  • Φορητότητας — να εξάγετε τα δεδομένα σας σε αρχείο JSON

Επειδή δεν διατηρούμε server με τα δεδομένα σας, η πλήρης διαγραφή γίνεται απευθείας από τη συσκευή σας, χωρίς να χρειάζεται αίτημα προς εμάς.
